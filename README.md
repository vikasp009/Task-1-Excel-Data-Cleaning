## Objective
To clean and standardize a real-world dataset using Excel and prepare it for further analysis.

## Dataset
Netflix Movies and TV Shows dataset (Kaggle)

## Tools Used
- Microsoft Excel

## Steps Performed
- Imported CSV data and verified column headers
- Froze header row and applied filters for exploration
- Identified and highlighted missing values
- Created a backup sheet before cleaning
- Checked and removed duplicate records using key identifiers
- Standardized text fields using TRIM, PROPER, and UPPER
- Validated numeric and categorical columns
- Handled inconsistent date formats by retaining date_added as text for consistency
- Created a separate Cleaned_Data sheet
- Added Data_Quality_Notes to document data issues

## Deliverables
- Raw_Data.xlsx
- Cleaned_dataset.xlsx
- cleaned_dataset.csv

## Outcome
A clean, structured dataset suitable for analysis, with documented data quality considerations.
